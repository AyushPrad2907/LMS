## Changes Made

### 1. Linked style.css in index.html
- **File:** `frontend/index.html`
- **Change:** Added `<link rel="stylesheet" href="style.css" />` to the head.
- **Reason:** The trial modal styles were defined in `style.css` but not loaded, causing the modal to be visible unstyled at the bottom of the page.
- **Revert:** Remove line 8 in `frontend/index.html`.

### 2. Standardized wildcard routes in server.js (REVERTED)
- **File:** `backend/server.js`
- **Change:** Reverted back to `{*any}` syntax.
- **Reason:** Express 5.x uses `path-to-regexp` v8, which requires named parameters for wildcards. The original `{*any}` was correct; changing it to `*` caused a `PathError`.
- **Revert:** No action needed (already reverted to functional state).

### 3. Improved error handling in trial routes
- **File:** `backend/routes/trial.js`
- **Change:** Added handling for Mongoose `ValidationError` to return `400 Bad Request`.
- **Reason:** Provides better feedback for validation failures (e.g. invalid phone numbers).
- **Revert:** Remove the `if (error.name === "ValidationError")` block.

### 4. Added logging to trial.js
- **File:** `frontend/trial.js`
- **Change:** Added `console.log` for the API response.
- **Reason:** Helps track form submission status in the console.
- **Revert:** Remove line 92 in `frontend/trial.js`.

### 5. Fixed lint warning in index.html
- **File:** `frontend/index.html`
- **Change:** Removed a duplicate `transition` property on `.cat-card`.
- **Reason:** Improved CSS quality.
